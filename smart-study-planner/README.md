# 🤖 AI Smart Study Planner

## 📌 Overview
This project generates a personalized study plan based on user subjects, weak areas, and available study time.

It intelligently prioritizes weak subjects and distributes time accordingly.

---

## 🚀 Features
- Accepts multiple subjects
- Identifies weak subjects
- Allocates more time to weak areas
- Generates smart study plan
- Provides study advice

---

## ⚙️ How to Run
1. Install Python
2. Run the program:

python app.py

---

## 🧠 Concept Used
This project simulates an AI-based recommendation system.

- Uses weighted logic to prioritize weak subjects  
- Distributes time based on importance  
- Mimics intelligent decision-making  

---

## 💡 Example

Input:
Subjects: math, physics, chemistry  
Weak subjects: math  
Hours: 6  

Output:
math → 3.0 hours  
physics → 1.5 hours  
chemistry → 1.5 hours
